```
プライベートIP: 10.238.2.39

プロトコル: OpenVPN (UDP 1194)
用途:
  - USMH社員・Luvina開発者からのVPNアクセス起点

セキュリティグループ:
  - UDP 1194: OpenVPN接続 (許可元IP制限あり)
```


## AMI バックアップ設定

```
推奨設定:
  - AWS Backup で週次 AMI スナップショット取得
  - EBS スナップショット: 毎日自動取得
  - 保持期間: 30日

現状確認コマンド:
aws ec2 describe-instances \
  --region ap-northeast-1 \
  --filters "Name=instance-state-name,Values=running" \
  --query 'Reservations[*].Instances[*].{Name:Tags[?Key==`Name`]|[0].Value,ID:InstanceId,Type:InstanceType,IP:PrivateIpAddress,AZ:Placement.AvailabilityZone}'
```

## インスタンス操作コマンド

```bash
# インスタンス一覧確認
aws ec2 describe-instances \
  --region ap-northeast-1 \
  --filters "Name=instance-state-name,Values=running,stopped" \
  --query 'Reservations[*].Instances[*].{Name:Tags[?Key==`Name`]|[0].Value,ID:InstanceId,State:State.Name,Type:InstanceType,IP:PrivateIpAddress}'

# インスタンス停止
aws ec2 stop-instances \
  --region ap-northeast-1 \
  --instance-ids <instance-id>

# インスタンス起動
aws ec2 start-instances \
  --region ap-northeast-1 \
  --instance-ids <instance-id>

# インスタンス再起動
aws ec2 reboot-instances \
  --region ap-northeast-1 \
  --instance-ids <instance-id>

# AMIスナップショット作成
aws ec2 create-image \
  --region ap-northeast-1 \
  --instance-id <instance-id> \
  --name "kasumi-backup-$(date +%Y%m%d)" \
  --description "Manual backup $(date)" \
  --no-reboot
```
